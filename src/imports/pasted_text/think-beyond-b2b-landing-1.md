Generate a high-fidelity B2B Landing Page wireframe for "Think Beyond", a higher education tech consulting company. The primary goal is lead generation targeting University Chancellors.

## Visual & Style Constraints:
- Use a STRICT grayscale color palette (Black, White, Light Gray, Dark Gray). No other colors. Focus entirely on layout, Information Architecture, and conversion.
- Style: Modern, clean, enterprise B2B, highly scannable, frictionless, and SOFT (rounded corners, pill-shaped buttons).
- Typography: Sans-serif, bold headlines, legible body text. High contrast.

## Mobile Responsiveness Guidelines (CRITICAL):
- All interactive elements must stack vertically (1-column stack) on mobile viewports.
- HERO SECTION FONT REDUCTION: On mobile screens, the main Hero Headline MUST be significantly reduced in font size.
- For ALL call-to-action buttons: width: 100% of the parent container with appropriate margins. Text must wrap dynamically if too long.
- Enforce generous padding and margins (at least 20px) between elements.
